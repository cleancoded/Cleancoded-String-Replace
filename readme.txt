== Description ==
String Replace replaces any defined string with another string within any WordPress website.

= Here are some common use cases for this tool =
* HTTPS rewriting (replace "http://domain.com" with "https://domain.com")
* WordPress Multisite domain mapping (replace any "myblog.domain.com" with "www.myblog.com")
* After migrating a website from a development server, dynamically update hardcoded instances of the dev URL
* Make changes to sitemap.xml (non-static files such as from `XML Sitemap Generator`) or to your robots.txt file
* Repace or change any content

= Please note =
* This tool can significantly slow down performance and page load times. If this tool is required, it should be used in combination with a caching plugin.

== Installation ==

1. Upload the `cleancoded-string-replace.zip` to the `/wp-content/plugins/` directory and unzip it
2. Activate the plugin through the 'Plugins' menu in WordPress
